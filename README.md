# Planner Studio Pro

A premium planner builder for creating, customizing, previewing, printing, and exporting professional planners.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Features

- **9 Planner Types**: Daily, Weekly, Monthly, Habit, Budget, Wellness, Fitness, Student, Business
- **Drag & Drop Builder**: Reorder blocks visually
- **Live Preview**: Real-time updates with zoom controls
- **Export Engine**: PDF (multi-page), PNG, JPG at up to 300 DPI
- **Template Gallery**: 12 pre-built templates
- **Color Palette System**: 8 preset palettes + custom colors
- **Settings**: Auto-save, grid, rulers, page size defaults

## Tech Stack

Vite + React 18 + TypeScript + Tailwind CSS + Zustand + React DnD + pdf-lib + Framer Motion
